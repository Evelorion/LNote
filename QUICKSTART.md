## ⚙️ 快速设置步骤

### 步骤1: 打开项目
1. 启动 **Android Studio**
2. 点击 **File** → **Open**
3. 选择 `d:\SoftWare\project\Note` 文件夹
4. 点击 **OK**

### 步骤2: 同步Gradle依赖
- Android Studio 会自动提示 "Gradle sync now"
- 点击 **"Sync Now"** 按钮
- 等待依赖下载完成（可能需要几分钟）

### 步骤3: 创建虚拟设备（模拟器）
如果没有实体手机，可以使用模拟器：

1. 点击工具栏的 **AVD Manager** 图标（设备管理器）
2. 点击 **Create Virtual Device**
3. 选择一个设备模型（如 Pixel 4a）
4. 选择 API level 32 或之后的版本
5. 完成创建

### 步骤4: 运行应用
1. 确保模拟器正在运行或设备已连接
2. 点击工具栏的 **Run** 按钮（绿色三角形）
3. 选择目标设备
4. 等待应用编译和安装

应用将在10-30秒内启动，您会看到采用iOS风格玻璃化UI的笔记本应用。

## 🎨 自定义iOS风格

### 修改玻璃效果透明度
编辑 `app/src/main/res/values/colors.xml`：
```xml
<!-- 调整透明度百分比（80 = 50%, B3 = 70%, 66 = 40%） -->
<color name="glass_white_light">#B3FFFFFF</color>
```

### 更改主题色
在 `colors.xml` 中更改 iOS 蓝色：
```xml
<color name="ios_blue">#FF007AFF</color>  <!-- 调整此值 -->
```

### 调整圆角半径
在布局文件中修改 `GlassContainerView`：
```xml
app:cornerRadius="24dp"  <!-- 增加或减少圆角 -->
```

### 更改应用名称
编辑 `app/src/main/res/values/strings.xml`：
```xml
<string name="app_name">我的笔记本</string>
```

### 更改应用图标
将自定义图标放置在：
- `app/src/main/res/mipmap-xxhdpi/ic_launcher.png`
- `app/src/main/res/mipmap-xxhdpi/ic_launcher_round.png`

## 🐛 常见问题解决

### 问题1: Gradle同步失败
**解决方案**:
- 检查网络连接
- 在 `gradle.properties` 中添加代理设置
- 使用离线模式重新构建

### 问题2: 编译错误 "Cannot resolve symbol"
**解决方案**:
- 点击 **File** → **Sync Now**
- 清除缓存: **File** → **Invalidate Caches/Restart**

### 问题3: 应用在设备上崩溃
**解决方案**:
- 查看 **Logcat** 日志（底部面板）
- 确保目标设备API级别 >= 21

### 问题4: 数据未保存
**解决方案**:
- 确保点击了 "保存" 按钮
- 检查 **Logcat** 中的数据库错误
- 尝试卸载并重新安装应用

### 问题5: 玻璃效果看不清楚
**解决方案**:
- 确保设备背景使用浅色
- 检查颜色透明度值（在 colors.xml 中）
- 在低端设备上可能效果有限
