## 🔧 自定义和扩展

### 更改主色调
编辑 `app/src/main/res/values/colors.xml`:
```xml
<color name="ios_blue">#FF007AFF</color>
```

### 调整玻璃效果透明度
编辑玻璃色定义:
```xml
<color name="glass_white_light">#B3FFFFFF</color> <!-- 70% -->
```

### 修改圆角
在布局中:
```xml
app:cornerRadius="16dp"
```

### 启用暗黑模式
创建 `values-night/colors.xml` 并定义深色版本

## 🐛 常见问题

**Q: 玻璃效果看不清楚？**
A: 检查背景颜色，浅色背景时效果最明显。参考 [iOS_GLASS_DESIGN.md](iOS_GLASS_DESIGN.md) 的调试部分。

**Q: 应用在低端设备上运行缓慢？**
A: 软件层渲染会消耗资源。考虑在硬件配置低的设备上关闭某些效果。

**Q: 如何支持暗黑模式？**
A: 在 `values-night` 目录下创建颜色文件，使用深色方案。详见 [iOS_UI_INTEGRATION.md](iOS_UI_INTEGRATION.md)。

**Q: 如何导出笔记？**
A: 目前不支持。可以扩展 `NoteDao` 添加导出功能。
